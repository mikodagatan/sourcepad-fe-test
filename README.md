# Cyanide is Happiness

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

### How to start

```
yarn # to install
yarn start
```

### Includes the following stack

- React
- Typescript
- Prettier
- TailwindCSS + PostCSS
- Recoil
